######## COMANDOS NPM #######

Criar projeto:
→ npx create-react-app . (caso queira usar com o nome da pasta que está usa-se o ponto ".", caso queira criar uma pasta com o nome, troca-se o ponto pelo nome desejado)

Iniciar aplicação no navegador:
→ npm start

Instalar aplicações de  desenvolvimento:
→ npm i eslint babel-eslint --save-dev

Configurar o Eslint:
→ npx eslint --init
